#一条动态的评论列表

##接口地址
/feeds/{feed_id}/comments

##请求方法
GET

##额外请求参数（get传入）
limit 请求数据条数  默认15条
max_id 用来翻页的记录id(对应数据体力的feed_digg_id)

|----------|:--------:|:--------:|:--------:|
| comment_id  | string      | yes      | 评论id |
| create_at | string	  | yes		 | 评论时间 |
| comment_content     | string  	  | yes 	 | 评论内容 |
| user     | array      | yes    | 评论者基本信息 |
| replyUser     | array      | yes    | 被评论者基本信息 |
