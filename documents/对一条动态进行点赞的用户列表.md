#对一条动态点赞的用户列表

##接口地址
/feeds/{feed_id}/diggusers

##请求方法
GET

##返回体
```json5
{
  "status": true,
  "code": 0,
  "message": "操作成功",
  "data": [
    {
      "name": "Wayne",
      "phone": "18908019700",
      "email": ""
    },
    {
      "name": "管理员",
      "phone": "18781993582",
      "email": ""
    }
  ]
}
```

##返回字段
| name     | type     | must     | description |
|----------|:--------:|:--------:|:--------:|
| name  | string      | yes      | 用户名称 |
| phone | string	  | yes		 | 用户手机号 |
| email     | string  	  | no 	 | 邮箱 |

code请参见[Feed消息对照表](Feed消息对照表.md)
