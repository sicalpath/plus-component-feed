#@我的动态列表

##接口地址
/api/v1/feeds/atme

##请求方法
GET

##额外请求参数（get传入）
limit 请求数据条数  默认10条
max_id 用来翻页的记录id(对应数据体力的atme_id)

##返回体
```json5
{
  "status": true,
  "code": 0,
  "message": "操作成功",
  "data": [
    {
      "atme_id": 1,
      "feed_id": 1,
      "feed_title": "1",
      "feed_content": "123123",
      "created_at": 1487011968,
      "user": {
        "user_name": "Wayne",
        "phone": "18908019700"
      }
    }
  ]
}
```

##返回字段
| name     | type     | must     | description |
|----------|:--------:|:--------:|:--------:|
| feed_id  | int      | yes      | 动态id |
| feed_title | string	  | yes		 | 动态标题 |
| feed_content     | string  	  | yes 	 | 动态内容 |
| created_at | timestamp    | yes      | 发布时间 |
| user_name | string    | yes      | 用户名 |
| phone | string    | yes      | 手机号 |

code请参见[Feed消息对照表](Feed消息对照表.md)
